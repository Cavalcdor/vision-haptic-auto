# vision-haptic-auto — Descripción de las Figuras

Este directorio contiene las figuras utilizadas en el artículo.

## Figuras Generadas

| Archivo | Contenido |
|---------|-----------|
| `fig0_architecture.png` | Diagrama de la arquitectura de retroalimentación de tres capas del sistema |
| `fig1_force_haptic_mapping.png` | Curvas de simulación del mapeo fuerza-sensación háptica (3 subgráficos) |
